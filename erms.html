Manual: Extreme Rainfall Monitoring System (ERMS)

Extreme Rainfall Monitoring System

The Extreme Rainfall Monitoring System (ERMS) is an experimental tool that provides estimated locations of potential or ongoing extreme rainfall and their possible impacts, based on the selected date using a 5-day forecast (GFS) and 5-day historical data (IMERG).

üî∏ Step 2: Map Render and Results Panel

(Refer: Figure 2)

  • After clicking the "Render" button, the system processes input and:
    • Displays rainfall and its intensity
    • Highlights likelihood of flooding and alert level
    • Activates IMERG and GFS Panels with result summaries: 'Affected Palikas with different alert levels'
    • Sectoral checkboxes appear for user interaction
This is the core visualization phase showing areas at risk and potential impacts.

üî∏ Step 3: Sectorial Results Update

(Refer: Figure 3)

  • When users select sectoral checkboxes, the Results Panel dynamically updates to show:
    • Vulnerable populations: elderly, disabled, and children
    • Exposure: crops, buildings, drinking water, toilets
This interactive step enables customized impact analysis based on user interest and concern.
